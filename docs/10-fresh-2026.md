# 10. Самое свежее (v40–v41, 2026)

Дайджест новейших релизов UEFN/Fortnite Ecosystem за 2026 год. Актуально на июнь 2026.
**Последний крупный релиз — v41.00 (5 июня 2026)**, ближайший патч v41.10 — 25 июня 2026.
(v42.00 на момент составления ещё не вышел.)

---

## 🤖 AI-инструменты (ключевое за 2026)

- **Epic Developer Assistant** — ИИ-ассистент прямо в редакторе: генерирует Verse-код
  (включая **Scene Graph**), отвечает на вопросы, даёт подсказки по F1-наведению.
  → https://dev.epicgames.com/community/assistant/fortnite
- **AI-Powered Conversations** (эксперимент, v40.20) — NPC на базе LLM: с характером, голосом,
  реагируют на действия игрока и влияют на геймплей. Это **другое**, чем Developer Assistant
  (тот — для разработчика, этот — для NPC в игре).

---

## v41.00 — 5 июня 2026

- **Sidekicks как скриптуемые NPC** — компаньоны-существа (монстро-баттлеры, фермы, collection-RPG). Публикация островов с 23 июля.
- **Custom Weapon Templates** — 4 базовых шаблона (Pistol, SMG, AR, Shotgun): свой меш/звук/VFX + баланс (урон, скорострельность, разброс, отдача).
- **Mobile dev tools (эксперимент)** — Touch Input Gestures API (tap/drag), кастомные тач-контролы через HUD Controller, тапаемые виджеты.
- **Skeletal Animation в Scene Graph (эксперимент)** — новый Verse API `PlaySkeletalAnimation` для SG-сущностей.
- **Перестройка Vehicle Verse API** — `GetOccupants()` → `GetPassengers()` + `GetDrivers()`; новые `AddAgent()`, `RemoveAgent()`, `GetSeats()`, интерфейс `fort_vehicle_seat`.
- **Verse-добавления** — `GetGlobalTransform`, `IsWeakpointDamage` у `damage_result`.
- **Player Movement device** — Roll Landing, Wall Kick, Jump Fatigue, Shoulder Bashing, Slide Kick.
- **Новые тулзы**: Session Inspector, Publishing Report, Memory Snapshot, валидация LOD0 (>30k вершин — ошибка), переделанная Gizmo System.

### ⚠️ BREAKING (v41.00)
- Неймспейс Scene Graph: `using { /UnrealEngine.com/SceneGraph }` **больше не компилируется** →
  используй **`/Verse.org/SceneGraph`**.
- Предупреждения о нелегальном override свойств теперь — **ошибки, блокирующие публикацию**.

---

## v40.40 — 14 мая 2026
- **Player Input API вышел из эксперимента (стабилен)** — управление активными инпутами игрока из Verse.
- Cube Rifle, Free Camera debug-команда, числовой ввод в Debug Command Menu.

## v40.20 — 16 апреля 2026
- **Custom Items & Inventory → Beta (можно публиковать)** — кастомные предметы, внутриигровые экономики, крафт.
- **AI-Powered Conversations (эксперимент)** — см. выше.
- Стандартизованы Verse-пути предметов: `/Fortnite.com/Items`, `/Fortnite.com/Weapons`.

## v40.00 — 19 марта 2026
- **Book of Verse** — новый официальный референс языка (публичный GitHub); старые страницы Verse Language Reference устаревают.
- **Verse-теги на сущностях** — находить помеченные объекты в рантайме без настройки полей/ссылок.
- **Star Wars toolset** — 25+ оружий, герои (Luke, Vader, Boba Fett), транспорт (X-wing, TIE, N-1). Публикация с 1 мая.
- **Physics для Carryable Spawner device**.
- **Scene Events v4** — `OnReceive`, новые типы spatial-math, экспериментальный `agent_group`.
- Миграция UI-инпутов: `/UnrealEngine.com/ControlInput/UI` → `/Verse.org/Input/UI`.
- Custom Items: `item_details_component`→`description_component`, `item_icon_component`→`icon_component`; новый `stackable_component`.
- Редактор: переделанный Home Panel, Branch Explorer для Revision Control (Beta).

---

## Что из этого важно для нас

1. **Scene Graph неймспейс сменился** → в нашем `verse/components/` используем `/Verse.org/SceneGraph` (уже обновлено).
2. **Custom Items вышли в Beta** — можно делать реальные внутриигровые экономики и крафт.
3. **ИИ-ассистент Epic** реально помогает писать Verse — стоит подключить.
4. **Мобилки** продолжают получать тач-API — приоритет Epic.
5. **Vehicle API переписан** — старый `GetOccupants()` мёртв, используем `GetPassengers()`/`GetDrivers()`.
